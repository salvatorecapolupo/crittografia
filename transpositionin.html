SIMPLE TRANSPOSITION INSTRUCTIONS

What is a Simple Transposition Cipher?

The Simple Transposition Cipher is a type of transposition cipher in which letters of the encrypted message are reversed.

How to Solve

When solving the Simple Transposition Cipher, the player is given an encrypted message and a cipher. To decrypt the message, the player reverses the order of the encrypted message so that when it is read backwards, it reveals the decrypted message.

Example

encrypted message: ereht olleh

decrypted message: hello there
